📌

วิธีตั้งค่า (ทำครั้งเดียว)

1. เปิด Google Sheets → สร้าง Spreadsheet ใหม่
2. เมนู ส่วนขยาย → Apps Script → วางโค้ดด้านล่าง → กดบันทึก (💾)
3. กดปุ่ม "ทำให้ใช้งานได้" (มุมขวาบน) → "การทำให้ใช้งานได้ใหม่"
   · ประเภท → เลือก "เว็บแอป"
   · ดำเนินการในฐานะ → "ฉัน"
   · ผู้ที่มีสิทธิ์เข้าถึง → "ทุกคน"
   · กด "ทำให้ใช้งานได้" → อนุญาตสิทธิ์ → คัดลอก URL
4. นำ URL มาวางด้านล่าง → กด บันทึก URL
5. กลับไปที่ Google Sheet → ใช้ฟอร์มด้านล่างเพิ่มรายชื่อเจ้าหน้าที่
